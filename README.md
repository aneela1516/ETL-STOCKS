# ETL-STOCKS
This is a Fintech based Stock Market ETL project using real-time data from yahoo finance. 

What this project covers: 
  1. Fetching real stock market data using `yfinance`
  2. Setting up a PostgreSQL database
  3. Building a Star Schema Data Model
  4. Writing an ETL Job (Extract → Transform → Load)
  5. Querying data programmatically using SQL + Python
  6. Visualizing results

Skills: 
  1. Python (Matplotlib, Pandas, SQLAlchemy, yfinance, psycopg2)
  2. SQL (DDL, DML, JOINS, GROUP BY, CASE WHEN, AGGREGATIONS)
  3. Data Modeling
  4. ETL Pipeline
  5. PostgreSQL
  6. Data Engineering
